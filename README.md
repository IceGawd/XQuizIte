## Getting Started  
To start working, first set up the repository locally.
```bash
git clone https://github.com/IceGawd/XQuizIte.git
cd xquizite
```
Check the [Wiki](https://github.com/IceGawd/XQuizIte/wiki) for further information on building in Unity.
